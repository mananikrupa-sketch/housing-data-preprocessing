# Housing Data Cleaning and Preprocessing

## Project Overview
This project focuses on cleaning and preprocessing housing affordability data using Python.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Preprocessing Steps
- Imported dataset
- Checked missing values
- Removed duplicates
- Renamed columns
- Performed outlier detection
- Created visualizations
- Categorical data to vector(encoding)
- Saved cleaned dataset

## Files
- housing_data.csv
- cleaned_housing_data.csv
- preprocessing.ipynb
